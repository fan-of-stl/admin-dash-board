# 🏆 Admin Dashboard (Atomic Design)

![Dashboard Screenshot](./assets/screenshot.png)

## 🚀 About The Project

This **Admin Dashboard** is built using **Atomic Design** principles, providing a modular and reusable component structure. It features an intuitive UI for managing data, tracking analytics, and visualizing insights.

## ✨ Features
✅ **Atomic Design Architecture**  
✅ **Reusable UI Components (MUI, Chart.js)**  
✅ **Dynamic Charts & Graphs**  
✅ **Data Tables for Management**  
✅ **Dark Mode Support (Optional)**  

## 🛠 Tech Stack
- **Frontend:** React, Material-UI, Chart.js  
- **State Management:** React Context (or Redux if added)  
- **Build Tool:** Vite
